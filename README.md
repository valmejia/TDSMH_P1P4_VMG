# TDSMH_P1P4_VMG

## NOTA IMPORTANTE SOBRE LAS PRÁCTICAS 4 y 5

Este repositorio  contiene la practica 4 y la practica 5. 
 - **Práctica 4:** Solo implementa la operación de **SUMA**.
 - **Práctica 5:** Implementa **TODAS las operaciones** (suma, resta, multiplicación y división), incluyendo también la suma.
     
## Descripción
Aplicación Android desarrollada en **Java** con **Android Studio** que:


1. **Primera pantalla (Menú de operaciones):** Muestra 4 botones:
   - Suma
   - Resta
   - Multiplicación
   - División

   Al hacer clic en cualquier botón, la app navega a la segunda pantalla.

2. **Segunda pantalla (Calculadora):** Muestra un formulario con:
   - Campo para el **primer número**
   - Campo para el **segundo número**
   - Botón para **calcular**
   - Resultado de la operación seleccionada

   Dependiendo del botón pulsado en el menú, la calculadora realizará la operación correspondiente.


## Cómo ejecutar el proyecto
1. Clona el repositorio:
   
   ```bash
   git clone https://github.com/valmejia/TDSMH_P1P4_VMG.git

2. Abre Android Studio.
3. Espera a que se sincronicen las dependencias (Gradle).
4. Conecta un dispositivo físico o inicia un emulador.
5. Haz clic en Run.
